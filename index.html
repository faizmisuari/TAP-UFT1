Welcome to UFT Test Project

This project tests the Login functionality of https://the-internet.herokuapp.com/ using UFT One 2022 Build 983

Getting Started

  1. Clone this repository to your local machine.
  2. Delete any a.txt files
  3. Open your UFT testing tool.
  4. Locate and open the UFT test script file.
  5. Configure any necessary settings or parameters.
  6. Run the UFT test to validate the functionality of the website.

For more detailed information, refer to the project's README file.
